Guia de Comandos Git e GitHub

git init

Inicializa um novo repositório Git no diretório atual. Cria uma pasta oculta .git que armazenará todo o histórico de versões do seu projeto.

git status

Mostra o estado atual do repositório, exibindo arquivos modificados, adicionados ou não rastreados.

git add

Adiciona arquivos à área de preparação (staging area). Exemplo: git add . adiciona todos os arquivos modificados.

git commit -m "mensagem"

Salva as alterações adicionadas no repositório local com uma mensagem descritiva.

git log

Exibe o histórico de commits realizados no repositório, incluindo autor, data e mensagem.

git branch

Lista as branches (ramificações) existentes ou cria uma nova branch se usado com um nome, por exemplo: git branch nova-feature.

git checkout

Permite alternar entre branches ou commits específicos. Exemplo: git checkout main muda para a branch principal.

git merge

Mescla as alterações de uma branch em outra. Exemplo: estando na main, use git merge nova-feature para unir as mudanças.

git remote add origin <url>

Conecta seu repositório local a um repositório remoto no GitHub. Exemplo: git remote add origin https://github.com/usuario/repositorio.git

git push -u origin main

Envia seus commits locais para o repositório remoto (GitHub). O -u define a branch main como padrão para futuros pushes.

git pull

Atualiza seu repositório local com as mudanças mais recentes do repositório remoto.

git clone <url>

Clona (baixa) um repositório existente do GitHub para o seu computador.

git status

Mostra o estado atual do repositório, exibindo arquivos modificados, adicionados ou não rastreados.

Exemplo:
$ git status

git commit -m "mensagem"

Cria um novo ponto no histórico do repositório local com uma mensagem descritiva.

Exemplo:
$ git commit -m "Adiciona nova seção ao site"

git merge

Mescla alterações de uma branch em outra. Pode gerar conflitos se houver mudanças no mesmo arquivo.

Exemplo:
$ git merge nova-feature
(Se houver conflitos, o Git pedirá para resolvê-los antes de concluir o merge.)
